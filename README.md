# Advent of Code 2021

Code solutions to [Advent of Code 2021](https://adventofcode.com/2021)

## Execute Challenges
Execute the solutions with the following command where `n` is the challenge day. 

`go run advent-of-code-2021/src/day<n>/d<n>.go`
